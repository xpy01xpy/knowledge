> 路由的编码 和 对应的解码
> encodeURI, decodeURI：用于编码整个URI,因为URI中的合法字符都不会被编码转换。
> encodeURIComponent，decodeURIComponent：在编码单个URIComponent（指请求参数）应当是最常用的，它可以将参数中的中文、特殊字符进行转义，而不会影响整个URL。
